# カクテルリストアプリ
[Swift]


お酒のリストと選択されたベースのカクテルの画像を表示するアプリです。

レンタルサーバーを用意し、画像URLを入れたJSONファイルと画像を格納しました。
そのJSONファイルを利用して、swiftでお酒のリストと画像表示をするアプリを作りました。

タブビューを使用して、WebViewとカクテルを表示する画面を切り替える構成です。
